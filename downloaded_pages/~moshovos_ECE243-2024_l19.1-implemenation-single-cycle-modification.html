     TMP = MEM[R2]

Datapath Modifications

 

Recall that the datapath should be able to perform all the actions necessary by any instruction. So what are the actions necessary for the new ADD instruction?

 

  1. We must be able to read an instruction from memory using the PC as the address
  2. We must be able to use field R2 from the instruction to specify a register to read from.
  3. We must be able to use the value read from the register file in step 2 and use it from memory to read a byte.
  4. We must be able to use field R1 from the instruction to specify a register to read from.
  5. We must be able to add the values read in steps 3 and 4.
  6. We must be able to use field R1 from the instruction to specify a register to write into the value generated in step 5.
  7. We must be able to update Z and N according to the value produced in step 5.
  8. Must be able to read PC, add 1 and write the result PC+1 back into the PC.
